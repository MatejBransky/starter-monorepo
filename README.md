# starter-lerna

Monorepo with support for multiple packages and one app.

## Features

- ES6 support
- HMR enabled
- ESLint enabled
- Prettier-Eslint enabled for `precommit`

## Requirements

- Node.js >= 6
- Yarn >= 1

## How to use

**Instal dependencies**

```
$ yarn install
```

**Start development:**

```
$ yarn start
```

It starts the server with app.

**Build all for production:**

```
$ yarn build
```

**Build packages:**

```
$ yarn build:pkg
```

**Build app:**

```
$ yarn build:app
```
